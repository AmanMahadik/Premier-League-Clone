⚽ Premier League Clone Website

A modern, responsive, and feature-rich Premier League-inspired football website built using HTML, CSS, Bootstrap, and JavaScript. This project recreates the look and feel of the official Premier League platform while showcasing football news, transfers, statistics, clubs, players, fixtures, and interactive fan content.

🚀 Features
🏆 Premier League-inspired UI/UX
📱 Fully responsive design for desktop, tablet, and mobile
🎨 Modern dark-themed interface with custom gradients and animations
📰 News & Features section with interactive cards
🔄 Hero news carousel with featured stories
💰 Summer Transfers section
📊 Statistics and league table pages
👥 Players and Clubs directory
🎮 Games, Quizzes & Polls section
🤝 Premier League Partners showcase
📂 Multi-page architecture (16+ pages)
🍔 Mobile sidebar navigation
✨ Sticky animated navbar on scroll
🎯 Bootstrap Icons integration
⚡ Optimized reusable CSS and JavaScript components
🛠️ Technologies Used

CSS3
Bootstrap 5
JavaScript (ES6)
Bootstrap Icons
