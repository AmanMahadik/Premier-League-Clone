## 🎯 About the Project

A **pixel-perfect, fully responsive** Premier League-inspired website that recreates the look, feel, and interactivity of the official platform. From live-style fixtures to the transfer window — every section is crafted with attention to detail.

> Built as a frontend showcase demonstrating real-world UI architecture across 16+ interconnected pages.

## 🚀 Getting Started

### Option 1 — Open directly in browser

```bash
# Clone the repo
git clone https://github.com/amanmahadik/Premier-League-Clone.git

# Navigate into the folder
cd Premier-League-Clone

# Open the homepage
open HTML/home.html
```

### Option 2 — Use Live Server (recommended)

```bash
# If you have VS Code + Live Server extension
# Right-click on home.html → "Open with Live Server"
```

> No build tools. No dependencies. No npm install. Just open and run. ✅

## 🎨 Design System

|-------|-------|-------|
| `--pl-purple` | `#38003c` | Primary brand / navbar |
| `--pl-green` | `#00ff85` | Accent / highlights |

| `--card-bg` | `#1a1a2e` | Card & section backgrounds |

## 🌟 Key Features Walkthrough

**🏠 Homepage** — Hero carousel with featured match stories, top news grid, and partner showcase.

**📅 Matches** — Fixture cards organized by gameweek with team crests, dates, and result badges.

**📊 Table** — Live-style league standings with position indicators, form guides, and sorting.

**💰 Transfers** — Summer window tracker with incomings, outgoings, and transfer fee display.

**🎮 Fan Zone** — Interactive polls, football quizzes, and prediction games.
